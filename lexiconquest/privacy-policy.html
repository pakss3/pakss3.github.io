Lexicon Quest 개인정보처리방침

시행일: 2025년 1월 1일

1. 개인정보의 수집 및 이용 목적

「Lexicon Quest」(이하 "앱")는 다음의 목적을 위하여 개인정보를 처리합니다.

  • 회원 관리: 이메일 주소를 통한 회원 가입, 본인 확인, 계정 복구
  • 학습 서비스 제공: 학습 진행 상황, 레벨, 경험치, 스트릭 등 게이미피케이션 데이터 저장
  • 서비스 개선: 앱 사용 패턴 분석을 통한 서비스 품질 향상
  • 리더보드 기능: 닉네임을 통한 주간 리그 순위 표시
2. 수집하는 개인정보 항목

앱은 서비스 제공을 위해 다음과 같은 개인정보를 수집합니다.

【필수 항목】

  • 이메일 주소 (회원 가입 시)
  • 닉네임 (게임 내 표시용)

【선택 항목】

  • 학년 정보 (맞춤형 학습 콘텐츠 제공)
  • 프로필 사진

【자동 수집 항목】

  • 학습 기록 (단어 학습 이력, 퀴즈 결과)
  • 게임 진행 데이터 (레벨, 경험치, 골드, 배지)
3. 개인정보의 보유 및 이용 기간

앱은 개인정보를 다음과 같이 보유합니다.

  • 로컬 저장: 앱이 기기에 설치되어 있는 동안 기기 내부에 안전하게 보관
  • 클라우드 동기화: 회원 가입 시 Firebase 서버에 암호화되어 저장
  • 회원 탈퇴 시: 서버에 저장된 모든 개인정보가 즉시 삭제됨
  • 앱 삭제 시: 기기에 저장된 모든 로컬 데이터가 삭제됨
4. 개인정보의 제3자 제공

앱은 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다.

다만, 다음의 경우에는 예외로 합니다.

  • 광고 서비스: Google AdMob을 통한 광고 제공 시 익명화된 광고 ID만 사용 (개인 식별 불가)
  • Firebase 서비스: Google Firebase를 통한 인증 및 데이터 저장 (Google 개인정보처리방침 적용)
  • 법령에 의한 요청: 수사기관의 법적 절차에 따른 요청이 있는 경우
5. 정보주체의 권리

이용자는 언제든지 다음의 권리를 행사할 수 있습니다.

  • 개인정보 열람: 설정 > 프로필에서 저장된 정보 확인 가능
  • 개인정보 수정: 닉네임, 학년 등 프로필 정보 수정 가능
  • 개인정보 삭제: 설정 > 데이터 초기화 메뉴를 통해 전체 데이터 삭제 가능
  • 회원 탈퇴: 설정 > 로그아웃 후 계정 삭제 요청 가능
  • 처리 정지: 앱 삭제를 통해 모든 데이터 처리 정지 가능
6. 개인정보의 안전성 확보 조치

앱은 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다.

  • 비밀번호 암호화: Firebase Authentication을 통한 안전한 비밀번호 해싱 처리
  • 전송 암호화: 모든 네트워크 통신은 TLS/SSL 암호화 적용
  • 저장 암호화: Firebase/Google Cloud의 서버측 암호화 (AES-256) 적용
  • 로컬 저장: Hive 데이터베이스를 통한 기기 내 안전한 저장
  • 접근 제한: 앱 데이터는 해당 앱에서만 접근 가능 (샌드박스 환경)
